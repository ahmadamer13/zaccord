How can I contribute as a software developer? • 1 open position

You can contribute on GitHub or, if you’re serious, join us on a project basis with flexible hours.

For the latter, you should meet these criteria:

  • Strong grasp of algorithms and data structures, especially for 3D objects
  • Experience in software and web development
  • Knowledge of JavaScript, Node.js, SQL/MySQL, and core web technologies
  • Confident with Git and GitHub; experience with open‑source workflows
  • Nice to have: 3D printing experience; C/C++ knowledge

If you make substantial contributions on GitHub or discover an accurate, efficient algorithm to price an STL print for given parameters without generating G‑code, we will take your application very seriously.

How can I contribute as a CAD/3D printing engineer? • No open positions

In that case, these skills are expected:

  • Hands‑on 3D printing experience
  • Solid knowledge of CAD, Fusion 360 or Blender
  • Experience in 3D modeling and design
  • Nice to have: electronics experience
  • Nice to have: C/C++ programming
